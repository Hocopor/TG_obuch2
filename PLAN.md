## Журнал решений

- **2025-07-15 — Стек**: Python + aiogram 3 (бот), FastAPI + Jinja2 + SQLAlchemy (админка), PostgreSQL (БД), Docker, Caddy.
- **2025-07-15 — Порты**: бот — фоновый процесс, админка — 5000, PostgreSQL — 5433.
- **2025-07-15 — Прокси**: SOCKS5 через aiohttp-socks, timeout=300 (int, не ClientTimeout).
- **2025-07-15 — Кэш**: file_id в таблице cached_files, автоматическое заполнение при первой отправке.
- **2025-07-15 — Видео**: .mov → .mp4 через ffmpeg, .mov в .gitignore.
- **2025-07-15 — Меню**: инлайн-кнопки (InlineKeyboardMarkup), не reply-кнопки.
- **2025-07-15 — Юр-документы**: загружаются через админку, отдаются по публичному URL.
- **2025-07-15 — Оплата**: на стороне GetCourse, бот только фиксирует.
